# 03 - Tests host local et multijoueur

## Objectif

Valider la capacité de la base Conan Exiles Enhanced / UE5 à fonctionner en host local avant l'ajout de mods.

## Host local solo

Test réalisé : environ 30 minutes en host local solo.

Résultat :

- aucun problème constaté ;
- pas de crash ;
- pas de blocage ;
- fluidité correcte.

Statut : **validé**.

## Test multijoueur réel Conan UE5

Le test avec un ami est actuellement différé faute de joueur disponible.

Statut : **différé, non bloquant**.

## Éléments favorables

- PC hôte solide : i9, 64 Go RAM, RTX 4070 12 Go.
- Connexion Internet Free 8 Gb/s stable.
- Historique d'hébergement multijoueur/coop régulier.
- Sessions déjà montées jusqu'à 7 joueurs sur Windrose.
- Objectif Conan plus modeste : 3 à 4 joueurs.
- Expérience passée d'hébergement Conan sous UE4.

## Conclusion actuelle

Le test multijoueur Conan UE5 reste à effectuer avant validation finale du modpack.

Cependant, il ne bloque pas la suite du projet.

Le risque principal ne semble pas venir de la machine ni de la connexion, mais plutôt de la stabilité du jeu UE5 et des futurs mods ajoutés.

## Règle future

Effectuer un test multijoueur réel :

1. dès qu'un ami est disponible ;
2. après les premiers mods installés ;
3. avant validation finale du modpack complet.
