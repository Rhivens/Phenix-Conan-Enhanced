# 07 - Mods Gameplay, Thralls & Animaux

## Objectif

Documenter les mods liés aux ajustements de gameplay, aux thralls, aux compagnons et aux animaux.

## Statut

**En cours.**

Mods Gameplay / Thralls actuellement validés :

1. **Better Thralls v3.0.0-enhanced**.
2. **Follower Remote v5.0.3 by Xevyr**.
3. **Xevyr's Hearthstone v5.0.0**.

## Critères de sélection

Les mods de cette catégorie devront :

- rester compatibles Conan Exiles Enhanced / UE5 ;
- ne pas déséquilibrer excessivement la progression ;
- éviter les modifications trop lourdes des systèmes centraux ;
- rester adaptés à une partie privée de 3 à 4 joueurs ;
- être testés prudemment, car cette catégorie peut impacter la stabilité et la sauvegarde.

## Mods validés

### Better Thralls v3.0.0-enhanced

- Catégorie : Thralls / compagnons / QoL gameplay.
- Source : Steam Workshop.
- Workshop ID : `3720904511`.
- Type : amélioration de la gestion des thralls, pets et followers.
- Statut : **validé en jeu**.

#### Résultat du test

- Abonnement Steam Workshop effectué.
- Mod activé depuis le menu Mods en jeu.
- Test réalisé en host local solo dans un pack avec Fashionist.
- Fonctionnement validé ingame.

#### Note de prudence

Better Thralls touche à des systèmes plus sensibles que les mods purement Interface & QoL.

Pour la suite, éviter d'ajouter immédiatement d'autres mods qui modifient lourdement les followers ou les limites de compagnons tant que la stabilité de ce bloc n'est pas mieux éprouvée.

#### Conclusion

**Better Thralls est validé comme premier mod du bloc Gameplay, Thralls & Animaux.**

### Follower Remote v5.0.3 by Xevyr

- Catégorie : Thralls / compagnons / QoL gameplay.
- Source : Steam Workshop.
- Workshop ID : `3718523921`.
- Type : gestion à distance du comportement des followers et thralls.
- Statut : **validé en jeu**.

#### Résultat du test

- Abonnement Steam Workshop effectué.
- Mod activé depuis le menu Mods en jeu.
- Test réalisé en host local solo dans le pack QoL avancé 10.6.
- Fonctionnement validé ingame.

#### Note de configuration

Réglages accessibles via :

```text
dc frsettings
```

ou via l'interface de réglages Xevyr quand elle est disponible.

#### Conclusion

**Follower Remote est validé comme second mod du bloc Gameplay, Thralls & Animaux.**

---

### Xevyr's Hearthstone v5.0.0

- Catégorie : Gameplay léger / mobilité / confort coop.
- Source : Steam Workshop.
- Workshop ID : `3719642461`.
- Type : pierre de retour liée au lit avec cooldown.
- Statut : **validé en jeu**.

#### Résultat du test

- Abonnement Steam Workshop effectué.
- Mod activé depuis le menu Mods en jeu.
- Test réalisé en host local solo dans le pack QoL avancé 10.6.
- Fonctionnement validé ingame.

#### Note de configuration

Réglage administrateur du cooldown via :

```text
dc hscooldown x
```

Le cooldown par défaut est à surveiller selon l'équilibre souhaité pour la partie coop.

#### Conclusion

**Xevyr's Hearthstone est validé comme mod de confort gameplay / mobilité.**
