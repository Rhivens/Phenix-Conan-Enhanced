# Mods à surveiller

## Objectif

Lister les mods potentiellement intéressants pour Conan Exiles Enhanced / UE5, sans les valider immédiatement.

## Règle

Un mod placé ici n'est pas encore retenu.

Il doit d'abord être vérifié sur les points suivants :

- compatibilité Enhanced / UE5 ;
- date de mise à jour ;
- commentaires récents des utilisateurs ;
- dépendances ;
- impact potentiel en host local ;
- risques de conflit avec les autres mods ;
- intérêt réel pour le groupe de joueurs.
